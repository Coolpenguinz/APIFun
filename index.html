I selected the Cat Fact API that gives the user random cat facts. I found it on RapidAPI.com. I used the getcatfact() and getcatfacts() methods. This API is useful if you are at a cat convention and want to amuse and impress people with your vast knowledge of the feline species. Additionally this API is useful if you need to entertain children for many hours. Pulling cat fact after cat fact out of your hat will prove invaluable in this setting.
